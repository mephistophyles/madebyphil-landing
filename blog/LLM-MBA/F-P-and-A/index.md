# Financial Planning & Analysis (FP&A): Comprehensive Syllabus

## Module 1: Introduction to FP&A and Financial Statement Analysis

**Core Concepts:**
- FP&A's role in organizational decision-making
- Key stakeholders and FP&A's relationship with other functions
- Financial statements overview: Balance sheet, income statement, cash flow statement
- Financial statement interconnections and articulation
- Common-size analysis and trend analysis
- Financial ratio categories: Profitability, liquidity, solvency, efficiency
- Benchmarking against industry standards
- Limitations of financial ratio analysis

**Lab Work/Problem Sets:**
1. Comprehensive ratio analysis of a public company compared to its top competitors
2. Creating dynamic common-size statements that update with new data inputs
3. Performing a DuPont analysis to decompose ROE drivers
4. Building an interactive financial dashboard using Excel/Power BI
5. Writing a financial health assessment memo for executive leadership

**Case Study:** "Detecting Financial Distress Signals" - Analyze historical financial statements of a company that experienced financial distress (e.g., JCPenney) to identify early warning signs that would have been visible to FP&A professionals.

## Module 2: Forecasting Techniques for Financial Planning

**Core Concepts:**
- Time series analysis fundamentals
- Sales forecasting methodologies: Moving averages, exponential smoothing
- Regression analysis for financial projections
- Seasonality adjustments and decomposition
- Top-down vs. bottom-up forecasting approaches
- Driver-based forecasting frameworks
- Sensitivity analysis and scenario planning
- Forecasting accuracy measurement techniques

**Lab Work/Problem Sets:**
1. Developing a 12-month revenue forecast using multiple methodologies (regression, time series)
2. Creating a driver-based expenses model with sensitivity toggles
3. Analyzing forecast accuracy using MAE, MAPE, and RMSE metrics
4. Building a Monte Carlo simulation for sales projections with probability distributions
5. Calibrating forecasting models based on historical accuracy
6. Implementing rolling forecast methodologies with variance analysis

**Case Study:** "Forecasting in High-Uncertainty Environments" - Examine how companies adjusted forecasting methodologies during the COVID-19 pandemic and developed new indicators for business tracking.
